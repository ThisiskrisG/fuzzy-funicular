# IVP Starter Bundle

This is the starter pack for IVP (Internet Visionary Provider). It includes:

- Frontend: `index.html`
- Backend: Flask app
- Q&A Plugin: `whisper_bot.js`
- Deploy scaffold
- Certs placeholder

## Standalone Terminal

You can launch the project components using the `ivp_terminal.py` script:

```bash
python ivp_terminal.py
```

Inside the terminal, type `start_backend` or `start_plugin` to run each service, `stop` to stop them, and `exit` to quit.
